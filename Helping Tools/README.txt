This folder contains the tools that were developed to help the preprocessing of the Makam Following Tool together with their reports.

The Half Automatic Pitch Track Smoother is tool for aiding the manual correction of the algorithm extracted pitch tracks.

Makam Information is a notebook for creating the Makam information dictionaries for Makam Hüseyni, Hüzzam, Saba, Rast.
It includes the perde names and interval lengths in cents. The idea can be used to create the dictionaries for the other makamlar.

The explanation on how they work and how they were created can be found on the Reports folder.
